Odoo - Product Management

Membuat Data Hazard Pictogram
  1. Buka menu Product -> Configuration -> GHS Hazard Information -> Hazard Pictogram
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Isi Hazard Statement. Harus diisi.
  2. Unggah gambar Image.
  3. Isi Code. Harus Diisi
  4. Ubah Active jika dibutuhkan.
  5. isi Description. Tidak harus diisi
  6. Klik tombol Save pada bagian atas-kiri form.
